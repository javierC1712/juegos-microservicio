## MICROSERVICIO DE JUEGOS

## Tecnologia usadas
**Java 21**
**SpringBoot (dependencias: JPA, VALIDATION, SPRING WEB, MYSQL, DEVTOOLS Y LOOMBOK)**
**Base De Datos: MySQL**

## PUERTO
**El microservicio esta alojado en el puerto localhost:7575/juegos**

**importante tener descargado el las extenciones: Extension Pack for Java y Spring Boot Extension pack, para poder conectarlo a la bd.**
